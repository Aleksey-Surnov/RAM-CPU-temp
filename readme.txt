RAM-CPU-temp serves for real-time monitoring of parameters:
- CPU temperature 
- involved RAM
- CPU load
You can save the results in a separate file after taking measurements.
The data file in csv format will be located in the same folder as the program
Use in operating systems: 
-Windows v.7,8,10 
-Linux Debian, Linux Mint, Linux Ubuntu.
-------------------------------------------------------------------------
Instructions:
THE PROGRAM MUST BE RUN AS ADMINISTRATOR!!!
Open file “read me”  and read the instruction
Run the application
Enter the language [Russian/English]
To start monitoring enter «yes», to exit the program enter «no».
Enter measurement time in minutes:
-you can enter integers of a series: 1, 2, 3…10,…123,….n
-you can enter decimal fractions of a series: 0.1, 0.15, 0.2……1.2, 1.35,….n.k
The program will output information about:
1.	Installed OS
2.	Name, frequency and number of processor cores
3.	Available physical RAM

During the specified time, data will be displayed in the terminal:
1.	Temperature CPU in C
2.	Loading CPU %
3.	Used RAM Gb

To save the results, enter “yes”, otherwise no, enter “no”
The result will be saved in the same folder as the program with the name “monitoring data (date) (time) .csv”
To repeat monitoring enter «yes»
To exit the program press the button «Enter»
